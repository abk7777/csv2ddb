# csv2ddb

Command line tool for easy loading of CSV files directly into AWS DynamoDB.

## Getting Started

Be sure to configure your AWS client:
```
aws configure
```
Install `csv2ddb`:

```
pip install csv2ddb
csv2ddb --help
```

### Prerequisites

Requires Python 3, Boto3 and the AWS CLI client.
